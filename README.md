# gtumedei.github.io

My personal website.

## Getting started

- Clone the repo
- Install dependencies (`pnpm i`)
- Create a `.env` file and populate it with the variables indicated by the schema under `src/lib/server/env.ts`
- Start development server (`pnpm dev`)
